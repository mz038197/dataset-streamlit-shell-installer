# Dataset Streamlit Shell

教學用 Streamlit 資料實驗室殼層：上傳／整理 CSV，並以內建範例走通監督式與非監督式演算法頁。

## Language

**K-近鄰分類**:
監督式分類教學頁的正式名稱；以距離找 k 個鄰居、多數決決定類別的實例型（懶惰）學習。側欄插在線性支持向量機與決策樹概念之間；僅使用內建範例資料（不提供 ready.csv 切換）。採訓練前預測殼（兩題解鎖＋Agent 提示），不做類神經網路式實驗迴圈。
_Avoid_: KNN 分類（可作副標／口語）、K 最近鄰、k-NN、KNN 回歸

**學習階段（K-近鄰分類）**:
頁內用 horizontal radio 切換的教學段落（不用 tabs）。階段1 標籤「鄰居與投票」：建立找鄰居＋多數決；階段2 標籤「選擇 k」：以 k 為主角（可帶尺度相關觀察）。各階段各自兩題訓練前預測。
_Avoid_: Tab、分頁（指 st.tabs 實作）；階段1 標籤出現「k／超參數」當主菜

**k（K-近鄰分類）**:
要查詢的最近鄰居個數。階段1 固定且不在 UI 露出（固定為奇數預設）；階段2 以奇數步進的 slider 調整。第一版距離固定歐氏、投票為多數決（uniform）；不露 weights／metric 進階項。
_Avoid_: 階段1 的 k slider（含 disabled 顯示）；第一版強調平票處理流程

**內建範例資料（K-近鄰分類）**:
兩特徵、二元分類的教學用 CSV。階段1／2 以可畫 2D 決策邊界的資料為主；階段2 另具備刻意尺度失衡的對照，供關閉標準化時觀察距離被某一軸主導。
_Avoid_: ready.csv 作為本頁資料來源；第一版 multiclass／高維主路徑

**訓練前預測（K-近鄰分類）**:
解鎖「開始訓練」前的概念關卡。階段1：實例型（無 w）與多數決；階段2：k 過大趨向過度平滑／多數類，以及未標準化時大尺度特徵主導距離。
_Avoid_: 類神經網路式實驗迴圈；硬鎖「必須先跟 Agent 說話才能訓練」

**查詢點（K-近鄰分類）**:
用來示範預測的單一座標點；畫出其 k 個最近訓練鄰居（可連線），讓「鄰居與投票」可見。以圖上 click 指定；本頁因此採用可點選的互動圖（Plotly），不沿用其他 ML 頁的 matplotlib＋st.pyplot 主路徑。
_Avoid_: 第一版以 confusion matrix／classification report 當主視覺；強制全專案改 Plotly
